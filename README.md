# Deloitte Australia Data Analytics Job Simulation

## Overview

This repository contains my work from the Deloitte Australia Data Analytics Virtual Experience Program hosted on Forage.

The simulation focused on solving real-world business problems using data analytics, visualization, and data classification techniques. The project involved analyzing manufacturing telemetry data to identify equipment downtime patterns and evaluating workplace pay equality using employee compensation data.

## Task 1: Telemetry Data Analysis In Tableau

### Business Problem

Daikibo Industrials collected telemetry data from manufacturing equipment across four global factories to understand machine health and identify operational inefficiencies.

The key questions were:

1. Which factory experienced the highest machine downtime?
2. Which machine types contributed most to downtime in that factory?

### Methodology

* Imported and structured telemetry data from multiple factories into Tableau.
* Created a calculated field named **Unhealthy** to quantify machine downtime.
* Built interactive visualizations showing:

  * Downtime per factory
  * Downtime per device type
* Developed an interactive dashboard with filtering functionality to enable drill-down analysis.

## Task 2: Gender Pay Equality Table

### Business Problem

Daikibo Industrials wanted to investigate potential gender pay inequality across factories and job roles using compensation data.

### Methodology

Created an additional classification column called **Equality Class** based on the Equality Score:

| Equality Score                    | Classification        |
| --------------------------------- | --------------------- |
| -10 to +10                        | Fair                  |
| -20 to -11 or +11 to +20          | Unfair                |
| Less than -20 or Greater than +20 | Highly Discriminative |
